What A Heap Is

A heap is a complete binary tree stored mostly as an array. It is used when you need fast access to the highest priority value: maximum in a max heap or minimum in a min heap.

O(1)peek root
O(log n)insert/delete
O(n)build heap
Read this in order: complete binary tree, array formulas, max heap vs min heap, heapify up, heapify down, build heap, insertion, deletion, then heap sort.

What You Should Read

  1. Complete binary tree: heaps are filled level by level from left to right.
  2. Heap property: max heap keeps the largest value at root; min heap keeps the smallest value at root.
  3. Array representation: no pointer tree is needed; index formulas connect parent and child.
  4. Heapify up: fixes insertion by moving a new value upward.
  5. Heapify down: fixes deletion/building by moving a wrong root downward.
  6. Heap sort: repeatedly moves max/min to final position.

High-Value Exam Points

Max heap Min heap Heapify Array indexing Insertion Deletion Priority queue No extra spaces
  • Output can have many valid heap arrangements, but online judges often expect the exact heap created by the given algorithm.
  • Build heap from index n / 2 - 1 down to 0.
  • For insertion, compare with parent (i - 1) / 2 until the new value reaches its correct place.
  • Heap is not a sorted array. Only the parent-child heap rule is guaranteed.

Max Heap Operations

  • Peek maximum: return arr[0].
  • Insert value: put value at the end, then heapify up while parent is smaller.
  • Delete maximum: replace root with last element, reduce size, then heapify down.
  • Build heap: call heapify down from n / 2 - 1 to 0.
Max heap insert condition:
while (i > 0 && arr[parent] < arr[i]) swap(parent, i);

Max heapify down:
choose the larger child and swap if child > parent.

Min Heap Operations

  • Peek minimum: return arr[0].
  • Insert value: put value at the end, then heapify up while parent is larger.
  • Delete minimum: replace root with last element, reduce size, then heapify down.
  • Build heap: same loop, but compare using smaller child.
Min heap insert condition:
while (i > 0 && arr[parent] > arr[i]) swap(parent, i);

Min heapify down:
choose the smaller child and swap if child < parent.

Heap Sort Idea

Heap sort uses heap structure to sort an array in-place. For ascending order, build a max heap, repeatedly swap the root with the last element, reduce heap size, and heapify the root.

  1. Build a max heap from the array.
  2. Swap arr[0] with the last element of the active heap.
  3. Reduce heap size by one because the largest element is now fixed.
  4. Heapify down from root.
  5. Repeat until the heap size becomes one.
Max heap gives ascending sorted output when sorted in-place. Min heap can be used for descending order with the same swap-to-end approach.

Dry Run Value

Input: 5, scores 3 1 4 2 5, new score 6.

  1. Build max heap from 3 1 4 2 5: result becomes 5 3 4 2 1.
  2. Insert 6 at the end: 5 3 4 2 1 6.
  3. Move 6 up because it is greater than parent 4: 5 3 6 2 1 4.
  4. Move 6 up again because it is greater than parent 5: 6 3 5 2 1 4.
Some solutions produce 6 4 5 1 2 3 depending on build order and swaps. Follow the exact algorithm your platform expects.

Array-Only C++ Code

#include <iostream>
using namespace std;

void swapValue(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

void maxHeapify(int arr[], int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }

    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }

    if (largest != i) {
        swapValue(arr[i], arr[largest]);
        maxHeapify(arr, n, largest);
    }
}

void buildMaxHeap(int arr[], int n) {
    for (int i = n / 2 - 1; i >= 0; i--) {
        maxHeapify(arr, n, i);
    }
}

void insertIntoMaxHeap(int arr[], int &n, int value) {
    arr[n] = value;
    n++;

    int i = n - 1;
    while (i > 0 && arr[(i - 1) / 2] < arr[i]) {
        swapValue(arr[i], arr[(i - 1) / 2]);
        i = (i - 1) / 2;
    }
}

int main() {
    int n;
    cin >> n;

    int arr[16];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    int d;
    cin >> d;

    buildMaxHeap(arr, n);
    insertIntoMaxHeap(arr, n, d);

    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << arr[i];
    }

    return 0;
}

Min Heap Code Changes

To convert max heap code into min heap code, change the comparisons. Pick the smallest child during heapify down, and move the inserted value upward while the parent is larger.

void minHeapify(int arr[], int n, int i) {
    int smallest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;

    if (left < n && arr[left] < arr[smallest]) {
        smallest = left;
    }

    if (right < n && arr[right] < arr[smallest]) {
        smallest = right;
    }

    if (smallest != i) {
        swapValue(arr[i], arr[smallest]);
        minHeapify(arr, n, smallest);
    }
}

void insertIntoMinHeap(int arr[], int &n, int value) {
    arr[n] = value;
    n++;

    int i = n - 1;
    while (i > 0 && arr[(i - 1) / 2] > arr[i]) {
        swapValue(arr[i], arr[(i - 1) / 2]);
        i = (i - 1) / 2;
    }
}

Common Mistakes

  • Forgetting to build the heap before inserting the new score.
  • Using wrong child formulas such as 2 * i for zero-based arrays.
  • Printing an extra space after the last number.
  • Sorting the array instead of maintaining heap structure.
