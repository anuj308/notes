Things to memorize tonight

  • Docker image vs container vs volume vs network.
  • Dockerfile: FROM, WORKDIR, COPY, RUN, CMD, ENTRYPOINT, EXPOSE, ENV.
  • Port mapping format: host_port:container_port.
  • Maven phases: validate, compile, test, package, verify, install, deploy.
  • GitHub Actions path: .github/workflows/file.yml.
  • Jenkins pipeline block: pipeline -> agent -> stages -> stage -> steps.
  • MySQL mandatory env variable: MYSQL_ROOT_PASSWORD.
  • Volume persistence: data survives container removal.

Exam execution formula

1. ReadIdentify technology, image, ports, env vars, files, and verification.
2. WriteOn paper: aim, required files, commands, expected output.
3. ExecuteCreate project, run commands carefully, avoid spelling mistakes in filenames.
4. VerifyUse docker ps, logs, curl/browser, mvn test, Actions/Jenkins console.
5. ExplainEnd with 2 viva points: why each command/step is needed.
